# IBM Data Analyst Capstone Project

## Objective
Analyze real-world datasets to identify trends and generate actionable business insights.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL
- Excel
- Cognos Analytics

## Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

## Key Outcomes
- Cleaned and transformed raw datasets
- Performed statistical and exploratory analysis
- Built dashboards to communicate insights

## Project Files

- dashboard screenshots → Visualization outputs
- report.pdf → Final project report
